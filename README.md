# 地质学大数据前沿与应用 —— 任务2：约束最优化法 CONOP

《地质学大数据前沿与应用》课程作业项目。樊隽轩教授，南京大学地球科学与工程学院。

## 任务概览

利用 CONOP（Constrained Optimization）方法进行定量地层对比，每人撰写 4-8 页小论文，格式参考《高校地质学报》；
小组准备 15 分钟 PPT 做课堂汇报。

## 可选主题

1. 数据收敛的影响因素
2. 如何判断是否接受当前解
3. 计算参数如何设置
4. 新的算法思路（高阶）

## 项目结构

```
.
├── CONOP-run/          # CONOP 运行数据（配置、结果、中间文件）
├── references/         # 参考文献 PDF
├── slides/             # 课堂课件（gitignored，不推送）
├── scripts/            # 分析脚本
├── 论文/               # 小论文草稿
└── PPT/                # 汇报 PPT
```

## 环境

- **运行 CONOP**：Windows 程序（需要台式机或 Wine）
- **数据分析**：Python 3（依赖见 requirements.txt）
